# 🌸 Iris Flower Classification using Machine Learning


## 📌 Project Description

This project classifies Iris flowers into different species using machine learning classification algorithms.

The species predicted are:

- Iris Setosa
- Iris Versicolor
- Iris Virginica


## 📊 Dataset Features

The model uses flower measurements:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width


## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn


## 🤖 Machine Learning Models

Algorithms used:

- Logistic Regression
- K-Nearest Neighbors (KNN)


## 📈 Model Evaluation

Performance checked using:

- Accuracy Score
- Confusion Matrix
- Classification Report


## 🚀 Workflow

1. Load Iris Dataset
2. Data Exploration
3. Data Visualization
4. Preprocessing
5. Train ML Models
6. Evaluate Performance


## 📌 Result

The trained model successfully classifies Iris flower species based on measurements.
