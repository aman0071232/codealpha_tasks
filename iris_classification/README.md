# 📌 Project Overview

The **Iris Flower Classification System** is a Machine Learning project that predicts the species of Iris flowers based on their physical measurements.

This project demonstrates the complete **classification workflow** used in Machine Learning including:

✔ Data Loading  
✔ Exploratory Data Analysis (EDA)  
✔ Data Visualization  
✔ Data Preprocessing  
✔ Model Training  
✔ Prediction  
✔ Model Evaluation  


The objective is to build an accurate classification model capable of identifying Iris flower species from measurement data.


---

# 🌺 Species Classified

The model predicts three different Iris flower species:

🌸 Iris Setosa  

🌸 Iris Versicolor  

🌸 Iris Virginica  


---

# 📊 Dataset Information

The famous Iris dataset contains flower measurement features used for classification.


## Input Features


| Feature | Description |
|-------|-------------|
| 🌱 Sepal Length | Length of flower sepal |
| 🌱 Sepal Width | Width of flower sepal |
| 🌸 Petal Length | Length of flower petal |
| 🌸 Petal Width | Width of flower petal |


## Target

Flower Species Classification:

- Setosa
- Versicolor
- Virginica
