# 📌 Project Overview

The **Car Price Prediction System** is an end-to-end Machine Learning project designed to predict the **estimated resale value of used cars** based on multiple automobile features.

This project follows a complete Machine Learning workflow including:

✔ Data Collection  
✔ Data Cleaning  
✔ Exploratory Data Analysis (EDA)  
✔ Feature Engineering  
✔ Model Training  
✔ Performance Evaluation  
✔ Web Application Deployment  


The goal of this project is to provide an intelligent system that helps estimate a fair market value of vehicles using historical data.

# 📊 Dataset Features

The model predicts car prices using:

| Feature | Description |
|-------|-------------|
| 💰 Present Price | Current market price of car |
| 🛣 Kilometers Driven | Total distance travelled |
| ⛽ Fuel Type | Petrol / Diesel / CNG |
| 🏢 Seller Type | Dealer / Individual |
| ⚙ Transmission | Manual / Automatic |
| 👤 Previous Owners | Number of previous owners |
| 📅 Car Age | Age of vehicle |

✔ Understands linear relationships between features

✔ Ensemble Learning Algorithm

✔ Reduces overfitting  
✔ Provides better accuracy  


---

# 🏆 Final Model Selected

## 🌲 Random Forest Regressor

Random Forest was selected as the final model because it achieved better performance and prediction accuracy compared to other models.


---

# 📈 Model Evaluation

The model performance was evaluated using:


### 📌 R² Score

Measures how well the model explains price variations.


### 📌 Mean Absolute Error (MAE)

Measures average prediction error.


### 📌 Root Mean Squared Error (RMSE)

Evaluates overall model prediction quality.
