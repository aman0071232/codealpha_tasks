# 📂 Projects Overview


## 🚗 Project 1: Car Price Prediction System


### 📌 Description

An intelligent Machine Learning application that predicts the resale price of a car based on various automobile features.

The project uses regression algorithms to analyze historical car data and estimate accurate selling prices.



### ✨ Features

✔ Interactive Web Application  
✔ Real-time Price Prediction  
✔ Modern Streamlit Dashboard  
✔ Machine Learning Backend  
✔ User Friendly Interface



### 📊 Input Features

- 🚘 Present Price
- 🛣 Kilometers Driven
- ⛽ Fuel Type
- ⚙ Transmission Type
- 🏢 Seller Type
- 👤 Number of Owners
- 📅 Car Age



### 🤖 Algorithms Used

- Linear Regression
- Random Forest Regressor 🌲


### 🏆 Final Model

Random Forest Regressor was selected because it provided better prediction accuracy.



### 📈 Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)



### 🚀 Deployment

The project is deployed using Streamlit Cloud.

🔗 Live Demo:https://ai--car-price-predictor.streamlit.app/

## 🌸 Project 2: Iris Flower Classification


### 📌 Description

A Machine Learning classification model that identifies Iris flower species using flower measurements.

The model classifies flowers into:

🌸 Setosa  
🌸 Versicolor  
🌸 Virginica



### 📊 Dataset Features

The model uses:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width



### 🤖 Algorithms Used

- Logistic Regression
- K-Nearest Neighbors (KNN)



### 📈 Model Performance

Model evaluation was performed using:

✔ Accuracy Score  
✔ Confusion Matrix  
✔ Classification Report



### Learning Outcomes

Implemented complete classification workflow:

1. Data Loading
2. Data Visualization
3. Preprocessing
4. Training
5. Prediction
6. Evaluation
